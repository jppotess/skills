# Zenith Loop Reference

The Spec-to-Ship loop is:

```text
spec
→ frozen source of truth
→ acceptance criteria
→ milestone plan
→ bounded implementation
→ deterministic validation
→ independent verification
→ gap review
→ replan
→ repeat
→ final evidence
```

Progress is measured by closing gaps against the frozen spec, not by whether an implementation pass says it is done.
