# .zenith State Template

Create these files at the current project root:

```text
.zenith/
  SPEC.lock.md
  acceptance_criteria.md
  plan.md
  state.md
  progress-log.md
  gap-log.md
  validators.md
  delivery-state.md
  final-report.md
```

Use `SPEC.lock.md` as the frozen source of truth. Use the other files to track acceptance criteria, milestones, current state, progress, open gaps, validation commands, delivery classification, and final evidence.
